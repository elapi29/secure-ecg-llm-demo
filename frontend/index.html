Rust + WebAssembly + secure metadata workflow

secure-ecg-llm-demo

A lightweight MVP for secure ECG metadata handling before AI or tool ingestion. Instead of passing metadata as a raw handoff, this demo signs and encrypts it in the browser, then verifies and decrypts it in a Rust backend.

ECG metadata JSON → sign in browser → encrypt in browser → send to Rust backend → verify → decrypt → recover structured payload
